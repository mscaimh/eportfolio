`install.packages("package_name")` - install package

`library(package_name)` - load package

`help("function")` or `?function` - shows information about the function

**foreign**
`read_sav("dataset.sav")` - to import SPSS files
`read_dta("dataset.dta")` - to import Stata files
`read_csv("dataset.csv")` - to import csv files

**readr**

`save(dataset, file = "data/dataset.rda")` - to save R files

`variable <- value` - variable assignment

`head(variable)` - gives first 6 values from the variable

`select()` - select variables (columns) in data by specifying their names

`filter()` - select cases (rows) in data by specifying their values

`arrange()` - order the cases in the data

`plot(y)` - new plot is shown under the Plots tab
